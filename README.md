# Machine_failure_prediction 🤖 
<br/>
• Built a supervised ML model to predict machine failures using sensor/operational data, performed end-to-end data cleaning, feature engineering and model training achieving 92% accuracy. 
<br/><br/>
• Deployed an interactive prediction interface using Streamlit, enabling real-time failure risk assessment with user
input parameters.
